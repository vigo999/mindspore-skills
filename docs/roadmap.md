# Roadmap

This roadmap is intentionally lightweight.

## Current focus

The current focus is to make core model training workflows deeper, more reliable, and more reusable, especially in single-machine or otherwise controlled environments.

Priority areas:

- readiness
- failure diagnosis
- accuracy debugging
- performance analysis
- migration and adaptation

## Next

Expand toward more complex training organization and broader workflow coverage, while keeping the public capability surface understandable.

Examples include:

- more complex training environments
- broader workflow integration
- richer diagnose patterns
- stronger example coverage

## Later

Explore broader model-workflow coverage beyond the current training-centered focus, including more advanced training stages and adjacent workflow needs.
